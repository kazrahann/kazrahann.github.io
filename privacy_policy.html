개인정보처리방침

최종 업데이트: 2026년 5월 25일

카즈라한 웍스(Kazrahann Works, 이하 "개발자")가 제공하는 모바일 게임 CardTrain(이하 "게임")은 이용자의 개인정보를 소중히 여깁니다. 본 방침은 게임 이용 과정에서 수집되는 정보와 그 처리 방법을 안내합니다.

1. 수집하는 정보

1-1. 광고 식별자

항목 Android 광고 ID (GAID)
수집 목적 맞춤형 광고 제공
처리 주체 Google LLC (AdMob) — 개발자가 직접 저장하지 않음

1-2. 인앱 구매 영수증

항목 구매 영수증 식별자
수집 목적 구매 내역 확인 및 고객 지원 대응
저장 방식 기기 내 로컬 저장 (앱 데이터 영역). 외부 서버로 전송되지 않음

1-3. 게임 진행 데이터

항목 덱 구성, 카드 목록, 게임 설정, 재화(젬·코인·블루큐브) 등
저장 방식 기기 내 로컬 저장 (PlayerPrefs). 외부 서버로 전송되지 않음

1-4. 게임 플레이 로그 / 분석 데이터 (향후 도입 예정)

아래 항목은 현재 수집하지 않으며, 향후 서비스 업데이트로 도입될 수 있습니다. 실제 도입 시점에는 본 페이지 상단의 업데이트 날짜가 갱신되며, 앱 내 공지로 안내합니다.

항목 게임 진행 통계(플레이 시간·진행도·승패·획득 보상), 사용 덱·차량 구성, 디바이스 식별자(설치 ID 또는 광고 ID), 디바이스 기종·OS 버전·언어 설정
수집 목적 게임 밸런스 조정, 서비스 안정성 개선, 이상 행위(부정 이용·매크로) 탐지, 신규 콘텐츠 기획에 필요한 통계 분석
처리 주체 개발자 직접 또는 Google Firebase(Google LLC) 등 분석 서비스
보관 기간 수집일로부터 최대 2년 또는 이용자가 삭제를 요청한 시점까지

1-5. 닉네임 및 공개 게임 기록 (향후 도입 예정)

아래 항목은 현재 수집하지 않으며, 향후 랭킹 시스템 도입 시 적용됩니다. 실제 도입 시점에는 별도 동의 절차를 거치며, 동의 거부 시 랭킹 등록 없이 게임을 계속 이용할 수 있습니다.

항목 이용자가 직접 입력한 닉네임, 랭킹 산정용 게임 기록(점수·진행도·클리어 시간 등)
수집 목적 랭킹 서비스 제공 및 타 이용자에게의 순위 공개
공개 범위 닉네임과 해당 게임 기록은 게임 내 랭킹 화면을 통해 다른 이용자에게 공개됩니다. 이용자는 닉네임 작성 시 실명·전화번호·이메일 등 개인을 식별할 수 있는 정보를 사용하지 않을 것을 권고합니다.
철회 / 삭제 게임 내 설정 또는 문의처 이메일을 통해 닉네임 변경·랭킹 제외 요청 가능

2. 제3자 서비스

게임은 아래 제3자 서비스를 사용합니다. 각 서비스의 개인정보 처리는 해당 서비스의 방침을 따릅니다.

서비스 제공사 목적 개인정보처리방침
Google AdMob Google LLC 광고 제공 보기
Google Play 결제 Google LLC 인앱 구매 처리 보기

3. 데이터 보유 기간

  • 광고 식별자: Google 정책에 따름
  • 인앱 구매 영수증·게임 진행 데이터: 앱 삭제 또는 앱 데이터 초기화 시까지 기기 내 보관
  • 게임 플레이 로그 / 분석 데이터 (도입 시): 수집일로부터 최대 2년 또는 이용자 삭제 요청 시점까지
  • 닉네임 및 공개 게임 기록 (도입 시): 랭킹 등록 해제 또는 이용자 삭제 요청 시점까지

4. 이용자의 권리

광고 맞춤 설정 거부 / 광고 ID 초기화

Android 기기에서 직접 설정할 수 있습니다.

  • Android 12 이하: 설정 → Google → 광고 → 광고 맞춤 설정 해제 또는 광고 ID 초기화
  • Android 13 이상: 설정 → 개인정보 보호 → 광고 → 광고 ID 삭제

게임 데이터 삭제

  • 전체 삭제: 기기 설정 → 앱 → CardTrain → 저장공간 → 데이터 삭제
  • 앱 삭제 시: 기기 내 저장된 모든 게임 데이터가 함께 삭제됩니다

데이터 삭제 요청

위 방법 외에 개인정보 삭제를 요청하시려면 아래 이메일로 문의해 주세요. 요청 접수 후 7영업일 이내에 처리합니다.

5. 아동 대상 서비스

게임은 만 14세 미만 아동을 대상으로 하지 않습니다.

6. 개인정보처리방침 변경

본 방침은 서비스 변경 또는 관련 법령 개정에 따라 업데이트될 수 있습니다. 경미한 변경(표현 수정·오탈자 정정 등)은 본 페이지 상단의 업데이트 날짜로만 안내하며, 수집 항목·이용 목적·보유 기간 등 이용자 권리에 중대한 영향을 미치는 변경은 시행일 최소 7일 전(이용자에게 불리한 변경의 경우 30일 전)부터 앱 내 공지 또는 앱 업데이트를 통해 사전에 안내하며, 필요한 경우 재동의 절차를 거칩니다.
