개인정보처리방침

최종 업데이트: 2026년 5월 6일

카즈라한 웍스(Kazrahann Works, 이하 "개발자")가 제공하는 모바일 게임 CardTrain(이하 "게임")은 이용자의 개인정보를 소중히 여깁니다. 본 방침은 게임 이용 과정에서 수집되는 정보와 그 처리 방법을 안내합니다.

1. 수집하는 정보

1-1. 광고 식별자

항목 Android 광고 ID (GAID)
수집 목적 맞춤형 광고 제공
처리 주체 Google LLC (AdMob) — 개발자가 직접 저장하지 않음

1-2. 인앱 구매 영수증

항목 구매 영수증 식별자
수집 목적 구매 내역 확인 및 고객 지원 대응
저장 방식 기기 내 로컬 저장 (앱 데이터 영역). 외부 서버로 전송되지 않음

1-3. 게임 진행 데이터

항목 덱 구성, 카드 목록, 게임 설정, 재화(젬·코인·블루큐브) 등
저장 방식 기기 내 로컬 저장 (PlayerPrefs). 외부 서버로 전송되지 않음

2. 제3자 서비스

게임은 아래 제3자 서비스를 사용합니다. 각 서비스의 개인정보 처리는 해당 서비스의 방침을 따릅니다.

서비스 제공사 목적 개인정보처리방침
Google AdMob Google LLC 광고 제공 보기
Google Play 결제 Google LLC 인앱 구매 처리 보기

3. 데이터 보유 기간

  • 광고 식별자: Google 정책에 따름
  • 인앱 구매 영수증·게임 진행 데이터: 앱 삭제 또는 앱 데이터 초기화 시까지 기기 내 보관

4. 이용자의 권리

광고 맞춤 설정 거부 / 광고 ID 초기화

Android 기기에서 직접 설정할 수 있습니다.

  • Android 12 이하: 설정 → Google → 광고 → 광고 맞춤 설정 해제 또는 광고 ID 초기화
  • Android 13 이상: 설정 → 개인정보 보호 → 광고 → 광고 ID 삭제

게임 데이터 삭제

  • 전체 삭제: 기기 설정 → 앱 → CardTrain → 저장공간 → 데이터 삭제
  • 앱 삭제 시: 기기 내 저장된 모든 게임 데이터가 함께 삭제됩니다

데이터 삭제 요청

위 방법 외에 개인정보 삭제를 요청하시려면 아래 이메일로 문의해 주세요. 요청 접수 후 7영업일 이내에 처리합니다.

5. 아동 대상 서비스

게임은 만 14세 미만 아동을 대상으로 하지 않습니다.

본 방침은 서비스 변경 또는 관련 법령 개정에 따라 업데이트될 수 있습니다. 중요한 변경 사항은 앱 업데이트 공지 또는 본 페이지 상단의 업데이트 날짜를 통해 안내합니다.
