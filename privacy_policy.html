개인정보처리방침

최종 업데이트: 2026년 6월 24일 · 버전: v1.2.1

카즈라한 웍스(Kazrahann Works, 이하 "개발자")가 제공하는 모바일 게임 CardTrain(이하 "게임")은 이용자의 개인정보를 소중히 여깁니다. 본 방침은 게임 이용 과정에서 수집되는 정보와 그 처리 방법을 안내합니다.

1. 수집하는 정보

1-1. 광고 식별자

항목 Android 광고 ID(GAID) 또는 iOS 광고 식별자(IDFA)
수집 목적 맞춤형 광고 제공 및 광고 효과 측정
처리 주체 Google LLC (AdMob) — 개발자가 직접 저장하지 않음
iOS 추적 동의 (ATT) iOS 14.5 이상에서는 앱이 IDFA에 접근하기 위해 별도의 추적 동의 팝업(App Tracking Transparency)이 표시됩니다. 이용자가 추적을 거부한 경우 IDFA는 수집되지 않으며, 비맞춤형 광고가 제공됩니다.

1-2. 인앱 구매 영수증

항목 구매 영수증 식별자
수집 목적 구매 내역 확인 및 고객 지원 대응
저장 방식 기기 내 로컬 저장 (앱 데이터 영역). 외부 서버로 전송되지 않음

1-3. 게임 진행 데이터

항목 덱 구성, 카드 목록, 게임 설정, 재화(젬·코인·블루큐브) 등
저장 방식 기기 내 로컬 저장 (PlayerPrefs). 외부 서버로 전송되지 않음

1-4. 게임 플레이 로그 / 분석 데이터 (향후 도입 예정)

아래 항목은 전체 이용자 대상 분석·밸런싱 목적으로는 현재 수집하지 않으며, 향후 서비스 업데이트로 도입될 수 있습니다. 실제 도입 시점에는 본 페이지 상단의 업데이트 날짜가 갱신되며, 앱 내 공지로 안내합니다. (개인별 클라우드 세이브 백업을 위한 누적 플레이 기록은 이미 수집 중이며 1-6절에 안내합니다.)

항목 사용 덱·차량 구성 추이, 디바이스 식별자(설치 ID 또는 광고 ID), 디바이스 기종·OS 버전·언어 설정
수집 목적 게임 밸런스 조정, 서비스 안정성 개선, 이상 행위(부정 이용·매크로) 탐지, 신규 콘텐츠 기획에 필요한 통계 분석
처리 주체 개발자 직접 또는 Google Firebase(Google LLC) 등 분석 서비스
보관 기간 수집일로부터 최대 2년 또는 이용자가 삭제를 요청한 시점까지

1-5. 닉네임 및 공개 게임 기록

랭킹 시스템에 자발적으로 참여하는 이용자에 한해 적용됩니다. 랭킹 참여 시 별도의 동의 절차를 거치며, 동의 거부 시 랭킹 등록 없이 게임을 계속 이용할 수 있습니다.

항목 이용자가 직접 입력하거나 자동 생성된 닉네임, 랭킹 점수, 통과 도시 수, 통과 노드 수, 마지막 목적지, 덱 카드 구성(카드 종류·레벨), 배치 차량 구성(차량 종류·획득 도시), 기관차 업그레이드 레벨
수집 목적 공개 랭킹 서비스 제공, 타 이용자의 전략 연구 및 커뮤니티 활성화
저장 위치 Google Firebase Firestore (Google LLC, 서울 리전)
공개 범위 닉네임과 게임 기록은 게임 내 랭킹 화면 및 웹 공개 게시판을 통해 누구나 열람 가능합니다. 닉네임에 실명·전화번호·이메일 등 개인을 식별할 수 있는 정보를 사용하지 않을 것을 강력히 권고합니다.
닉네임 정책 랭킹 참여 시 최초 설정한 닉네임은 이후 앱 내에서 변경할 수 없습니다. 변경이 필요한 경우 문의처 이메일로 요청하실 수 있습니다.
계정 삭제 요청 시 삭제 요청 시 닉네임은 즉시 "(은퇴한 기관장)"으로 익명화됩니다. 점수·순위 등 통계 수치는 리더보드 역사성 보존을 위해 익명 상태로 유지될 수 있습니다.

1-6. 클라우드 세이브 및 플랫폼 계정 연동

기기에 Google Play Games(Android) 또는 Game Center(iOS)가 로그인되어 있는 경우에 자동으로 적용됩니다. 위 1-5의 공개 랭킹 기록과는 별개로, 본인만 조회 가능한 비공개 백업입니다.

항목 소유 카드 목록(종류·레벨), 덱 구성, 젬 보유량, 완료한 튜토리얼 여부, 이번 시즌 최고 점수, 누적 플레이 통계(플레이·클리어·게임오버 횟수, 누적 획득 골드·다이아·재화, 카드 획득·합성 횟수, 전투 데미지, 적·중간보스·보스 처치 수 등)
수집 목적 앱 재설치 또는 기기 변경 시 동일 플랫폼 계정으로 게임 데이터 복원
연동 방식 게임 내 익명 계정이 기기의 Play Games/Game Center 계정에 자동으로 연동됩니다. 별도의 동의 화면 없이 게임 이용을 위한 최초 약관 동의 시 함께 적용됩니다.
저장 위치 Google Firebase Firestore (Google LLC, 서울 리전) — 본인 계정에서만 조회 가능
연동 거부 방법 게임 내에서 별도로 비활성화하는 기능은 제공하지 않습니다. 연동을 원하지 않으시면 기기에서 Play Games/Game Center 로그인을 해제한 상태로 게임을 이용해 주세요.

3. 데이터 보유 기간

  • 광고 식별자: Google 정책에 따름
  • 인앱 구매 영수증·게임 진행 데이터: 앱 삭제 또는 앱 데이터 초기화 시까지 기기 내 보관
  • 게임 플레이 로그 / 분석 데이터 (도입 시): 수집일로부터 최대 2년 또는 이용자 삭제 요청 시점까지
  • 닉네임 및 공개 게임 기록: 삭제 요청 시 닉네임은 즉시 익명화. 점수·순위 등 통계 수치는 역사성 보존을 위해 익명 상태로 유지될 수 있음
  • 클라우드 세이브(플랫폼 계정 연동): 플랫폼 계정 연동이 유지되는 동안 보관되며, 연동이 해제되거나 삭제를 요청하면 삭제됨

4. 이용자의 권리

광고 맞춤 설정 거부 / 광고 ID 초기화

각 기기에서 직접 설정할 수 있습니다.

  • Android 12 이하: 설정 → Google → 광고 → 광고 맞춤 설정 해제 또는 광고 ID 초기화
  • Android 13 이상: 설정 → 개인정보 보호 → 광고 → 광고 ID 삭제
  • iOS — 앱 추적 거부 (ATT): 설정 → 개인정보 보호 및 보안 → 추적 → "앱이 추적 요청하도록 허용" 비활성화
  • iOS — Apple 광고 맞춤화 거부: 설정 → 개인정보 보호 및 보안 → Apple 광고 → 개인 맞춤형 광고 끄기

게임 데이터 삭제

  • Android — 전체 삭제: 기기 설정 → 앱 → CardTrain → 저장공간 → 데이터 삭제
  • iOS — 전체 삭제: 기기 설정 → 일반 → iPhone(iPad) 저장 공간 → CardTrain → 앱 삭제 (iOS는 앱 데이터만 별도 삭제하는 기능을 제공하지 않으므로 앱을 삭제해야 합니다)
  • 앱 삭제 시: 기기 내 저장된 모든 게임 데이터가 함께 삭제됩니다

데이터 삭제 요청

위 방법 외에 개인정보 삭제를 요청하시려면 아래 이메일로 문의해 주세요. 요청 접수 후 7영업일 이내에 처리합니다.

5. 아동 대상 서비스

게임은 만 14세 미만 아동을 대상으로 하지 않습니다.

6. 개인정보처리방침 변경

본 방침은 서비스 변경 또는 관련 법령 개정에 따라 업데이트될 수 있습니다. 경미한 변경(표현 수정·오탈자 정정 등)은 본 페이지 상단의 업데이트 날짜로만 안내하며, 수집 항목·이용 목적·보유 기간 등 이용자 권리에 중대한 영향을 미치는 변경은 시행일 최소 7일 전(이용자에게 불리한 변경의 경우 30일 전)부터 앱 내 공지 또는 앱 업데이트를 통해 사전에 안내하며, 필요한 경우 재동의 절차를 거칩니다.
